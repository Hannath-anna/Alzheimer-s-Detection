About NeuroScan

Research-grade dual-modality Alzheimer's detection system

NeuroScan is a dual-modality research system combining clinical biomarker ensemble ML with MRI brain scan CNN. It features SHAP explainability via manual TreeSHAP path-tracing, ensemble voting across Random Forest, Gradient Boosting, and SVM, and Grad-CAM heatmaps showing exactly which brain regions drove each prediction.

SHAP Explainability
Manual TreeSHAP path-tracing generates per-patient waterfall charts showing which biomarkers pushed the prediction up or down — transparent, not a black box.
Ensemble Voting
Three independent models — Random Forest, Gradient Boosting, SVM — each cast a vote. Majority consensus reduces bias and improves robustness over any single model.
Grad-CAM Heatmaps
Gradient-weighted Class Activation Maps overlay a color heatmap on MRI scans — revealing the exact brain regions that drove the EfficientNetB0 CNN's decision.
